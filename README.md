# jtelb

### last update ( 1.0.5 ): 
 - You can write functions, middleware, replies, and keyboard modularly!

## installation
```
npm install jtelb
```


## usage
```
const bot = require('jtelb')
bot.define_bot('bot token', 'welcome message ( start message text )')
```

*start project and enjoy develop!*
